# Support

How to get help with Cyber Legion projects.

| Channel | Use for |
|---|---|
| Security issues | [security@cyberlegion.io](mailto:security@cyberlegion.io) — see [SECURITY.md](SECURITY.md) |
| General questions | [office@cyberlegion.io](mailto:office@cyberlegion.io) |
| Bug reports | Open an issue on the affected repository |
| Commercial inquiries | [office@cyberlegion.io](mailto:office@cyberlegion.io) |
| Web | [cyberlegion.io](https://cyberlegion.io) |

## What to include in a bug report

- The affected repository and version / commit SHA
- What you expected to happen
- What actually happened
- Steps to reproduce
- Relevant logs, error messages, or screenshots
- Your environment (OS, browser, runtime version where applicable)

## What we do not support

- Private forks of our code outside the org
- Reverse-engineered builds
- Unauthorised redistribution
